# Azure Resource Group Deployment using Terraform

## Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform to deploy an Azure Resource Group.

## Technologies

* Microsoft Azure
* Terraform

## Project Structure

* provider.tf
* main.tf
* variables.tf
* terraform.tfvars
* outputs.tf

## Resources Created

terraform init

terraform apply
